# 博客系统 – 产品规格文档

## 1. 愿景与目标
- **愿景**：打造一个现代化、高性能的个人技术博客平台，对标 CSDN 等主流技术社区，但更专注于个人品牌的构建与极致的书写，阅读体验。
- **目标**：
  1. 提供全功能的后端管理后台，支持高效的内容创作与管理。
  2. 构建 SEO 友好、移动端响应式的前端展示页面。
  3. 支持丰富的主题切换与互动功能（评论、点赞、收藏）。
  4. 实现容器化部署，确保在互联网环境下的稳定访问。

## 2. 角色画像
| 角色 | 核心需求 |
|--------|------------|
| **博主（管理员）** | 高效的 Markdown 编辑器、文章版本管理、草稿箱、定时发布、数据统计。 |
| **读者** | 快速加载、代码高亮、暗色模式、多端适配、评论互动、内容搜索。 |

## 3. 核心功能
### 3.1 后端管理系统
- **文章管理**：
  - **创建/编辑**：支持 Markdown 实时预览，代码块高亮，图片拖拽上传。
  - **上传md格式文章**：支持上传md格式文章，自动转换为文章保存发布。
  - **发布控制**：立即发布、定时发布、存为草稿。
  - **版本控制**：自动保存历史版本，支持一键回滚。
  - **删除与恢复**：软删除机制，支持从回收站恢复。
- **分类与标签**：多级分类管理，标签增删改查。
- **评论管理**：查看、审核、回复、删除评论。
- **仪表盘**：访问量统计、热门文章排行、近期评论概览。

### 3.2 前端展示系统
- **阅读体验**：
  - **响应式设计**：完美适配桌面、平板、手机。
  - **主题切换**：内置明亮/暗黑模式，支持自定义主题色。
  - **SEO 优化**：SSR（服务端渲染）或 SSG（静态生成），自动生成 Sitemap，Meta 标签动态配置。
- **互动功能**：
  - **评论**：支持嵌套回复、表情包。
  - **社交**：点赞、收藏、一键分享（微信、微博、Twitter）。
- **内容发现**：
  - **搜索**：全文检索，支持按标题、标签、内容搜索。
  - **导航**：文章目录（TOC）、面包屑导航、相关文章推荐。

### 3.3 部署与运维
- **容器化**：提供 `Dockerfile` 和 `docker-compose.yml`，一键部署。
- **网络访问**：配置 Nginx 反向代理，支持 HTTPS，确保公网访问安全稳定。

## 4. 非功能需求
- **性能**：首屏加载时间 < 1s，Lighthouse 评分 > 90。
- **安全性**：防 SQL 注入、XSS 攻击，后台登录支持验证码/MFA。
- **可维护性**：前后端分离，代码遵循 RESTful 规范，模块化设计。

## 5. UI/UX 指南
- **设计风格**：极简主义，突出内容，使用大留白与清晰的排版。
- **交互细节**：按钮悬停微交互，页面切换平滑过渡，加载骨架屏。
- **字体**：优先使用系统字体栈（San Francisco, Inter, Roboto），代码块使用 Fira Code 或 JetBrains Mono。
   - 增加评论、点赞、搜索功能，优化 SEO 与移动端体验。
3. **阶段三：高级特性**
   - 版本回滚、定时发布、主题切换、Docker 部署支持。

## 8. 数据模型（简述）
- `User`: 管理员账户信息。
- `Article`: 标题、内容、摘要、状态（草稿/发布）、发布时间、浏览量。
- `Category`: 分类名称、描述。
- `Tag`: 标签名称。
- `Comment`: 内容、作者、关联文章、父评论 ID。
- `Like/Favorite`: 用户与文章的关联记录。
