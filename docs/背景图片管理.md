# 背景图片管理指南

## 当前配置

背景图片配置文件位于：`apps/spa/src/config/heroBackground.ts`

### 默认设置
- **切换间隔**: 10秒（10000毫秒）
- **图片数量**: 5张星空主题图片
- **过渡效果**: 1秒淡入淡出

## 如何添加自定义背景图片

### 方法1: 直接编辑配置文件

打开 `apps/spa/src/config/heroBackground.ts`，在 `images` 数组中添加图片URL：

```typescript
images: [
  'https://example.com/my-image.jpg',  // 网络图片
  '/images/my-local-bg.jpg',            // 本地图片
  // ... 其他图片
]
```

### 方法2: 使用编程接口

```typescript
import { addCustomBackground } from '@/config/heroBackground';

// 添加网络图片
addCustomBackground('https://example.com/starry-sky.jpg');

// 添加本地图片（需先放到 public/images 文件夹）
addCustomBackground('/images/custom-background.jpg');
```

## 使用本地图片

1. 在项目根目录创建 `apps/spa/public/images` 文件夹
2. 将图片文件放入该文件夹
3. 在配置文件中使用 `/images/your-image.jpg` 引用

## 调整切换间隔

编辑 `heroBackground.ts` 中的 `interval` 值：

```typescript
interval: 15000,  // 改为15秒
```

或使用编程接口：

```typescript
import { updateInterval } from '@/config/heroBackground';

updateInterval(15000);  // 15秒
```

## 移除背景图片

```typescript
import { removeBackground } from '@/config/heroBackground';

removeBackground(0);  // 移除第一张图片
```

## 推荐图片来源

- **Unsplash**: https://unsplash.com/s/photos/starry-sky
- **Pexels**: https://www.pexels.com/search/night%20sky/
- **Pixabay**: https://pixabay.com/images/search/starry%20night/

## 图片要求

- **分辨率**: 建议 1920x1080 或更高
- **格式**: JPG、PNG、WebP
- **大小**: 建议 < 1MB（优化加载速度）
- **主题**: 星空、夜景、梦幻风格

## 性能优化

- 所有图片会在页面加载时预加载
- 使用 Framer Motion 的 AnimatePresence 实现流畅过渡
- 建议不超过10张背景图片以避免过长的预加载时间
